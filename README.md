# Gemi Coach Financiero

Gemi Coach Financiero es una aplicación que proporciona análisis financiero y asesoramiento utilizando la API de Gemini.  Analiza transacciones, ingresos y gastos para ofrecer un resumen completo de tu situación financiera.

## ¿Por qué usarlo?

- Obtén un análisis completo de tus transacciones financieras.
- Recibe asesoramiento personalizado basado en tus ingresos y gastos.
- Integración con la potente API de Gemini.
- Facilita la comprensión de tu situación financiera.
- Mejora tu toma de decisiones financieras.

## Tecnologías Utilizadas

- Python
- Flask
- Gemini API

## Instalación y Uso

1. Clona el repositorio: `git clone <URL_DEL_REPOSITORIO>`
2. Instala las dependencias: `pip install -r requirements.txt` (Asumiendo un archivo requirements.txt)
3. Configura las variables de entorno en `key.env` (basado en la presencia de este archivo)
4. Ejecuta la aplicación: `python main.py`


## Estructura del Proyecto

```
Gemi Coach Financiero/
├── .gitignore
├── key.env
├── main.py
├── models.py
```

- **Configuración de Algorand y Gemini:** Se configura la conexión con estas plataformas.
- **Análisis de transacciones:**  El código obtiene, clasifica y analiza las transacciones.
- **Generación de análisis con Gemini:** Se utiliza la API de Gemini para generar el análisis financiero.

# Gemi Financial Coach

Gemi Financial Coach is an application that provides financial analysis and advice using the Gemini API. It analyzes transactions, income, and expenses to offer a comprehensive summary of your financial situation.

## Why use it?

- Get a complete analysis of your financial transactions.
- Receive personalized advice based on your income and expenses.
- Integration with the powerful Gemini API.
- Facilitates understanding of your financial situation.
- Improves your financial decision-making.

## Technologies Used

- Python
- Flask
- Gemini API

## Installation and Usage

1. Clone the repository: `git clone <REPOSITORY_URL>`
2. Install dependencies: `pip install -r requirements.txt` (Assuming a requirements.txt file)
3. Configure environment variables in `key.env` (based on the presence of this file)
4. Run the application: `python main.py`
